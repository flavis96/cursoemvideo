Diferença entre tags semânticas (com significado) e tags de forma ou também chamadas não-semânticas (sem significado)

Tags como <b> são usadas apenas para dar forma e não tem semântica já a tag <strong> tem semântica, exemplos:

Esse trecho é usando a tag <b>

Esse trecho é usando a tag <strong>

Visualmente ambos são iguais, mas em termos de acessibilidade e entendimento do próprio HTML ou também do Google o de cima só altera a forma do texto enquanto o de baixo, altera e também da significado.

Outro exemplo disso é a tag <i> pra itálico que não é semântico e a tag <em> que é ênfase que é semântico.

Também com a tag <u> que é sublinhado que não é semântico e a tag <ins> que é inserido que é a forma semântica.

Listas ordenadas e Listas não ordenadas

Usando a tag <ol> podemos criar uma lista ordenada, por exemplo:

Lista de produtos:

  1. Produto 1
  2. Produto 2
  3. Produto 3
  4. Produto 4

Podemos também mudar o tipo usado para ordenar a lista usando o parâmetro 'type' com valores como '1' 'A' 'a' 'I' 'i' e podendo também escolher em qual número começar usando o parâmetro 'start' com valores de '1' até 'infinito', por exemplo:

Lista de coisas:

  1. Coisa 1
  2. Coisa 2
  3. Coisa 3
  4. Coisa 4

Usando a tag <ul> podemos criar uma lista não ordenada, por exemplo:

Lista de produtos:

  • Produto 1
  • Produto 2
  • Produto 3
  • Produto 4

Podemos também modificar a forma usada antes dos produtos usando o parâmetro 'type' com valores como 'circle' 'square' 'disc', por exemplo:

Lista de coisas.

  • Coisa 1
  • Coisa 2
  • Coisa 3
  • Coisa 4

Também pode misturar os tipos de listas e fazer lista dentro de lista.

Exemplo

  1. Snes
    • Super Mario World
  2. Playstation
    • God Of War
  3. Megadrive
    • Sonic
